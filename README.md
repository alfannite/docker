## 🚀 Overview

Docker Helper is a collection of installation and maintenance scripts designed to simplify Docker deployment on Linux servers.

The goal of this repository is to provide a clean, repeatable, and beginner-friendly way to install Docker using the official Docker repository while keeping the process easy to maintain and troubleshoot.

---

## ✨ Features

* Interactive installation wizard
* Official Docker repository
* Docker Engine installation
* Docker Compose plugin
* Docker Buildx plugin
* Docker service management
* Clean uninstall script
* Beginner-friendly workflow
* Homelab ready

---

## 📖 Documentation

Each operating system contains its own documentation and installation scripts.

Example:

```text
ubuntu/
├── install.sh
├── uninstall.sh
└── README.md
```

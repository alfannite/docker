## 🚀 Quick Start

Clone Repository

```bash
git clone https://github.com/alfannite/docker.git
cd docker
```

---

## 📦 Ubuntu

Run Installer

```bash
cd ubuntu
chmod +x install.sh
sudo ./install.sh
```

Remove Docker

```bash
cd ubuntu
chmod +x uninstall.sh
sudo ./uninstall.sh
```

## 📖 Features

* Interactive Installation Wizard
* Official Docker Repository
* Docker Compose Plugin
* Docker Buildx Plugin
* Docker Group Management
* Installation Verification
* Clean Uninstall Mode
* Complete Uninstall Mode
